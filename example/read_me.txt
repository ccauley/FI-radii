After installing WITio, download this folder to run an example illustrating how to name your spectral point files and image files consistently for this script to work. 
The zipped (7z file) contains the WIP file for this example. Unzip the 7z file using a program like Keka (mac OS) or 7-Zip (windows). 
Open MATLAB (m file) and run. The example loops through three example fluid inclusions.
